# A/B Testing Analysis

## A/B Testing Element
With a hypothesis, a success metric, and a variant cohorting system in place, you can run experiments, collect the data, and analyze the outcomes using SQL.

## Benefits
+ Hone instinct about your users over time
+ Getting good at defining success by matching data with user needs
+ Getting good at succeeding by matching design with user needs
